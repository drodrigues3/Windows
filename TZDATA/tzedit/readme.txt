Tzedit 1.0.0.0
Time Zone Editor

Description:
Tzedit is a freeware utility that allows you to edit time zone
information in Windows. All parameters can be edited such as
time zone name and information, time zone location on the world
map (MapID), Microsoft Time Zone Index, MUI strings for time
zone name and Dynamic DST settings. This utility works on any
version of Windows from Windows 95 to Windows 8.

Benefits:
1) Edit time zone information on any version of Windows;
2) Edit time zone location on the world map (MapID);
3) Edit Microsoft Time Zone Index (Index and IndexMapping);
4) Edit MUI strings for time zone name;
5) Edit time zone Dynamic DST settings;
6) Enjoy XP-like interface when editing
time zone information on Windows 9x/NT/2000;
7) View internal useful for developers data.

System requirements:
Windows 95/98/Me/NT/2000/XP/2003/Vista/2008/7/2012/8 Desktop

Installation:
Download archive from http://www.southsoftware.com/tzedit.zip,
unzip it anywhere on your hard drive and start tzeditx32.exe file.
If you are running a 64-bit Windows, you can also start
tzeditx64.exe file instead of tzeditx32.exe.
